Обновить Запустить тесты Загрузить XML тест-кейз

Что сюда писать: Поехали!
Пример:

testsuite=login_test,messages_test xml_testsuite=demo_xml target_url=topface.com browser=firefox

Возможные параметры:

testsuite: через запятую без пробелов перечисляем питоновские тестсьюты которые хотим запустить.
Возможные значения: login_test,marks_test,messages_test,profile_test

xml_testsuite: через запятую без пробелов перечисляем xml-тестьюты, которые хотим запустить

target_url: имя ресурса, который хотим тестировать. например topface.com или alpha.topface.com и так далее

browser: браузер, в котором выполнятся все тесты. если браузер не указан, тесты будут выполнены согласно браузер-маппингу, который указан на сервере
Возможные значения: firefox | ie | opera | chrome

По окончании работы, приложение выдаст alert-окно с надписью OK
